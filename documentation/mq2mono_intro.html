What is MQ2Mono?

In short its a plugin that allows you to run C# (+other languages) scripts inside of the MQNext Proces.

Why would you care?

For the developer
  • "Real" programming language, with full access to external libraries. Web Servers, Database connections, Networking libraries, etc.
  • Real time debugging!
  • Can run as much as you want per frame, or even in a different thread if you feel up to the challenge
